Forms Basic form directives for the needs of common E-Governement applications.

Basic example

Individual form controls automatically receive some global styling by adding form-input attribute. All textual input,textarea and select elements are set to width: 100%; by default.

Labels are added by the attribute form-label with the representing i18n-message-key. The alignment of the label is by default above the input field. By adding .form-inline or .form-horizontal to the form element, the position of the element is set automatically to the desired alignment.
